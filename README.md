# DNB Radiodiagnosis — Final Theory Revision

A structured, exam-focused revision repository for the **DNB Radiodiagnosis Final Theory** examination
(4 papers x 100 marks; 10 short-note questions of 10 marks each per paper).

> **Status: Paper I COMPLETE** — all five categories built to the full standard (full reading
> prose + complete written answers + visually-verified PMC images):
> **Musculoskeletal** (10), **Head & Neck** (7), **Neuroradiology** (10), **Endocrine** (3),
> **Breast** (2) = 32 topics. **Papers II-IV to follow** (tracker not yet scored for them).

## How this repo is organised

```
paper-1-musculoskeletal/
  README.md                 <- index + frequency weighting
  01-skeletal-trauma/
    reading.md              <- teaching material
    questions.md            <- DNB-style questions + model-answer pointers
  02-bone-tumours-approach/
  ...
```

Each topic folder has two files:

- **reading.md** — concise teaching notes built for the 10-mark short-note format
  (classification/enumeration first, then modality-wise findings, differentials, pearls, and what to draw).
- **questions.md** — practice questions in DNB subjective format with mark splits (e.g. `[4+6]`),
  including **real previously-asked NBE questions** where available, each with a model-answer skeleton.

MSK is simultaneously the **highest-weighted** Paper I category and the **largest knowledge gap**, so it returns the most marks per hour of study.

## How to use

1. Read `reading.md` for a topic.
2. Close it and attempt the questions in `questions.md` under time (a 10-mark note ~ 1 page / ~7 min).
3. Check against the model-answer pointers.
4. Pair with image plates from a standard text (Grainger & Allison / Sutton) — these notes are scaffolding, not a substitute for looking at films.
