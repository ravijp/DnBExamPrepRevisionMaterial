## Notes

- **No real NBE past-paper citations are recorded for these topics**, so every
  `questions.md` "Previously asked" section says so explicitly and all questions are
  marked as practice — nothing is fabricated.
- Images are PMC open-access figures, downloaded and visually verified (see each
  topic's `images/images.md`). A few high-yield diagrams (e.g. JNA Holman-Miller,
  ameloblastoma soap-bubble) had no clean open-source match in this pass and are
  described in the prose / "What to draw" instead — add your own plates.
- CT and MRI dominate head-and-neck imaging; the reading notes state where plain
  radiography / ultrasound / nuclear medicine have a limited or specific role.
