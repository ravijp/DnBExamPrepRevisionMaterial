## Notes

- **No real NBE past-paper citations are recorded for these topics** — each `questions.md`
  states this and all questions are practice; nothing is fabricated.
- Images are PMC open-access figures, downloaded and visually verified (see each topic's
  `images/images.md`). Pure schematic diagrams (e.g. intra- vs extra-axial sign cartoon,
  abscess-vs-GBM ring schematic) had no clean photographic source and are described in the
  prose / "What to draw" instead.
- CT and MRI dominate neuroradiology; advanced MR (DWI/ADC, DSC perfusion, MR spectroscopy,
  SWI) is emphasised where it changes management (e.g. abscess vs tumour, stroke selection,
  glioma grading).
